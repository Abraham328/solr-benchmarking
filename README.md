### [apachesolr-benchs]()

Benchmarking Script for running Stress tests to your site

### Requirements

Python

### Installation 

    $ wget https://github.com/julianromerajuarez/apachesolr-benchs/archive/master.zip
    $ unzip master.zip
    $ cd apachesolr-benchs
    
    $ python graphs.py
